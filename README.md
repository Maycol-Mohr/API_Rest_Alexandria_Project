# API_Rest_Alexandria Project

Project developed in the Java course at Trybe school.

Technologies used:

* Java 17;
* SpringBoot;
* MySql;
* Spring Web;
* Spring Data JPA;
* Postman;
* Spring Initializr;
* Git;

Developed in layers such as:

* Entities;
* Repositories;
* Services;
* Dtos;
* Controllers;
